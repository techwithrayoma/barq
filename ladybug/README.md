
# LadyBug ✿


## Requirement • • •


### Step 1: Create virtual environment
```bash
conda create -n ladybug python=3.11 -y
```

### Step 2: Activate it
```bash
conda activate ladybug
```

### Step 3: Go to the folder
```bash
cd ladybug
```

### Step 4: Install requirements
```bash
pip install -r requirements.txt
```
